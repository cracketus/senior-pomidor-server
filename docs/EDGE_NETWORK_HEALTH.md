# Edge Network Health - Public Status Output

## Overview
Sanitized edge network health metrics exposed via public status endpoint.

## Metrics
- Latency (p50, p95, p99)
- Error Rate: % failed requests
- Throughput: RPS
- Availability: Uptime %

## Endpoint
GET /api/v1/status/edge-health
